# Gestion-Scolarite

A MERN web application for the tuition management in a school (ESI)

# How To setup :

First make sure you have NodeJS ^v12.13 installed on your machine

## working on Client-side

    *Open a terminal.
    *cd client
    *npm install
    *start working
    *push your work

## Working on Server-side

    *Open terminal.
    *cd server
    *npm install
    *start working
    *push your work
